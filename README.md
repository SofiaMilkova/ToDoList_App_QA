“Мои задачи”

Простое графическое приложение для управления списком задач.
Позволяет:
Добавлять задачи
Удалять задачи
Отмечать задачи как выполненные
Автоматически сохранять задачи в task.json

Используемые технологии
Python - логика приложения
Tkinter - интерфейс
json - сохранение данных

Для запуска ToDoList.py требуется Python 3.8+ 
Или открывать ToDoList.exe 

Функциональные требования
Пустая задача не должна добавляться в список
После добавления задачи она появляется в списке задач
Пользователь может удалить выбранную задачу
При удалении выбранная задача исчезает из списка
Пользователь может отметить задачу как выполненную
Выполненная задача помечается галочкой
Задача не должна отмечаться как выполненная несколько раз
Все задачи сохраняются автоматически в “task.json”
При повторном запуске программы список задач загружается из файла
Интерфейс программы должен корректно отображаться (все кнопки видны и активны).
